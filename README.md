# American Option Pricing & Volatility Modeling

## Overview
This project investigates the pricing of American put options under different volatility frameworks, combining quantitative finance models with data-driven approaches.

The study compares pricing accuracy, computational efficiency, and model behavior across volatility regimes.

---

## Key Features

- Implemented **Binomial Tree models** for American option pricing  
- Applied **Black-Scholes model** as benchmark  
- Developed **Heston stochastic volatility model** for dynamic volatility modeling  
- Constructed **Neural Network surrogate models** to approximate option prices  
- Evaluated model performance using **MAE and RMSE across volatility buckets**

---

## Methodology

- Simulated option pricing under:
  - Constant volatility (Black-Scholes)
  - Stochastic volatility (Heston)
  - Data-driven approximation (Neural Networks)

- Conducted:
  - Error analysis across volatility regimes  
  - Computational efficiency comparison  
  - Model robustness evaluation  

---

## Results

- Neural networks significantly improved pricing speed with competitive accuracy  
- Heston model captured volatility clustering effects better than Black-Scholes  
- Model performance varied significantly across volatility regimes
