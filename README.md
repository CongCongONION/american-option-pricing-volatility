# American Option Pricing under Volatility Structures

## Overview
This project studies American put option pricing under:
- Black-Scholes model
- Heston stochastic volatility model
- Market-implied volatility surface

## Key Work
- Built PDE-based pricing model for American options
- Compared BS vs Heston pricing differences
- Constructed implied volatility surface using AAPL options data
- Performed Heston calibration
- Developed neural network surrogate pricer

## Results
- Heston better captures volatility smile
- Pricing differences increase under high volatility
- Neural network significantly improves pricing speed

## Tech Stack
Python, NumPy, SciPy, Matplotlib, PyTorch
